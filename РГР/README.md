🎮 Сетевая игра "СТАРТАП"

Мультипоточная сетевая игра для креативных собеседований!
Работодатель vs Соискатели в фантастических проектах.
[ [ [
📋 Задание

Разработка сетевого приложения «Сетевая игра» с мультипоточной реализацией сервера.
Соединение клиентов через POSIX сокеты и C++ threads.
🛠 Используемые библиотеки
Библиотека	Назначение
iostream	Ввод/вывод данных
cstdio / stdio.h	Стандартный C I/O
string	Работа со строками
vector	Динамические массивы
cstring / string.h	Строковые операции C
unistd.h	POSIX API (сокеты, sleep)
algorithm	Алгоритмы STL
sys/types.h	Системные типы
sys/socket.h	Работа с сокетами
netinet/in.h	IPv4 адресация
netdb.h	DNS операции
time.h	Работа со временем
arpa/inet.h	IP функции
thread	Многопоточность C++

Быстрый запуск
1. Сервер (создание игры)

bash
make game

Сервер выведет порт — запишите его!
2. Клиент (подключение игрока)

bash
make player

Введите IP (localhost) и порт сервера

Примечания:

    Запускайте в разных терминалах

    Сервер автоматически выбирает порт

    Поддержка нескольких игроков

О игре "СТАРТАП"

Настольная игра для креативных собеседований в фантастических проектах!
Комплектация
Компонент	Количество
Карты «Профессий»	50 шт.
Карты «Навыков»	70 шт.
Карты «Эмоций»	50 шт.
Игроки	3–6 человек
Таймер	1 минута
Правила игры
Цикл раунда (по кол-ву игроков)

    Назначение ведущего
    Автоматически выбирается работодатель

    Профессии ведущему
    3 карты из колоды «Профессии»

    Презентация проекта
    Описание фантастического проекта + вакансии

    Раздача соискателям

        1 карта «Эмоции»

        4 карты «Навыки»

    Собеседование (по очереди)
    Шаг	Действие
    Шаг	Действие
    5.1	Выбор должности
    5.2	Презентация (≤1 мин)
    5.3	Вопросы от игроков
    5.4	Вскрытие карт
    5.5	Оценки 1-5 (сумма баллов)

    Распределение вакансий

        Профессия = +3 бонусных очка

        Профессия сохраняется

    Завершение раунда
    Использованные карты → на дно колоды

Конец игры

Игра завершается, когда каждый побывал ведущим
Победитель

Максимум баллов за все раунды!
