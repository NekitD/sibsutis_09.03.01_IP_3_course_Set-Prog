# Сетевая игра "СТАРТАП"

## ЗАДАНИЕ

Разработка сетевого приложения «Сетевая игра». Мультипоточная реализация сервера с установлением соединения с использованием библиотеки `thread` (C++).

ОБ ИГРЕ

«СТАРТАП» — настольная игра, создающая нестандартные и весёлые условия для проведения креативных собеседований.

    С одной стороны — работодатель, который набирает персонал в свой фантастический необычный проект

    С другой стороны — соискатель, у которого игрой заданы характер и необычное поведение

    В процессе игры все участники успевают побывать и в роли работодателя, и в роли соискателя.

КОМПЛЕКТАЦИЯ
Компонент	Количество
Карты «Профессий»	50 шт.
Карты «Навыков»	70 шт.
Карты «Эмоций»	50 шт.
Игроки	3–6 человек
Таймер	1 минута
ПРАВИЛА ИГРЫ
Структура игры

Игра состоит из нескольких этапов (по количеству игроков). Каждый этап — один раунд собеседований.
Цикл игры

    Назначение ведущего
    Автоматически выбирается ведущий (работодатель), остальные игроки становятся соискателями вакансий.

    Получение профессий ведущим
    Ведущий получает 3 карты из колоды «Профессии».

    Презентация проекта
    Ведущий описывает игрокам свой фантастический проект и вакансии, которые ему требуются (список вакансий дублируется под описанием).

    Раздача карт соискателям
    Каждый соискатель получает:

        1 карту из колоды «Эмоции»

        4 карты из колоды «Навыки»

    Собеседование (для каждого соискателя по очереди)
    Шаг	Действие
    5.1	Игрок выбирает одну понравившуюся должность
    5.2	Игрок презентует себя работодателю и объясняет, почему его нужно взять на работу (максимум 1 минута)
    5.3	Остальные игроки (включая работодателя) задают вопросы соискателю; соискатель отвечает
    5.4	Карты, которые были у соискателя, вскрываются
    5.5	Остальные игроки (включая работодателя) выставляют оценку от 1 до 5. Оценки суммируются в баллы, которые игрок получает за текущий этап

    Распределение вакансий

        Работодатель выбирает игроков, которым достанутся вакансии

        Каждая полученная профессия приносит +3 бонусных очка

        Профессия остаётся с игроком и может использоваться как заслуга на следующих собеседованиях

    Завершение раунда

        Использованные навыки и эмоции перемешиваются и отправляются на дно колоды

КОНЕЦ ИГРЫ

Игра заканчивается, когда каждый игрок побывал ведущим.
ПОБЕДИТЕЛЬ

Побеждает тот, кто набрал больше всех баллов в ходе игры.
