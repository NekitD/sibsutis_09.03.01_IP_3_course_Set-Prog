    ОБ ИГРЕ:
    "СТАРТАП" - настольная игра, создающая нестандартные еселые условия для проведения креативных собеседований.
С одной стороны - работодатель, который набирает в свой фантастический необычный проект персонал, с другой
стороны соискатель, у которого игрой заданы характер и необычное поведение.
    В процессе игры все участники успеют побывать и в роли работодателя, и в роли соискателя.

Комплектация:
50 карт "Профессий".
70 карт "Навыков". 
50 карт "Эмоций".
3-6 игроков.
Таймер на 1 минуту.

Правила:
Игра состоит из нескольких этапов (по количеству игроков).

Цикл игры выглядит следующим образом:

1) Автоматически выбирается ведущий (работодатель), а остальные игроки становятся соискателями вакансий.
2) Ведущий получает 3 карты из колоды "Профессии".
3) Ведущий описывает игрокам свой фантастический проект и вакансии, которые ему для этого проекта требуются 
    (список вакансий продублируется под описанием).
4) Каждый соискатель получает по одной карте из колоды "Эмоции" и по 4 карты из колоды "Навыки"
5) Далее по очереди для каждого соискателя выполняются следующие действия:
    5.1) Игрок выбирает понравившуюся должность (только одну).
    5.2) Игрок презентует себя работодателю и объяняет, почему именно его нужно взять на работу 
        (макс время собеседования 1 минута).
    5.3) Остальные игроки (включая работодателя) задают вопросы соискателю, на которые он отвечает.
    5.4) Карты, которые были у соискателя вскрываются.
    5.5) Остальные игроки (включая работодателя) выставляют соискателю оценку от 1 до 5. 
        Оценки суммируются в баллы, которые игрок получает за текущий этап игры.
6) Работодатель выбирает игроков, которым достанутся вакансии. 
    Каждая полученная профессия 3 бонусных очка и остаётся с игроком. 
    Он может использовать её как заслугу на следующих собеседованиях.
7) Использованные навыки и эмоции перемешиваются и отправляются на дно колоды.

Игра заканчивается, когда каждый игрок был ведущим.
Побеждает тот, кто набрал больше всех баллов в ходе игры.
